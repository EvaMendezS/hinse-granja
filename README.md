# 🐔 Hinse — Granja Avícola
### Sistema PWA de Control para Gallinas Ponedoras

App web progresiva (PWA) instalable en celular, funciona 100% offline.

## Módulos
- 📊 Dashboard con KPIs y gráficos
- 🐣 Ingreso de lotes y recría
- 🥚 Registro de postura diaria
- 🌾 Alimentación y costos
- 💉 Vacunación con alertas
- 💊 Medicación
- 📋 Mortandad

## Tecnología
HTML + CSS + JS puro · Chart.js · localStorage · Service Worker
